### 판매자 서비스

#### 상품 등록 및 관리 Part.

1. 상품 등록 <br/>
  A. 판매자의 상품 등록 <br/>
      • 카테고리 선택, 상품명, 가격, 재고수량, 상품 이미지, 상세 설명 <br/>
      • 썸머노트를 적용해서 다양한 태그와 이미지를 드래그해서 다중 이미지 삽입 가능 <br/>
  B. 상품 목록 조회 <br/>
      • 상품 이름을 클릭하여 등록된 상품을 수정할 수 있습니다.<br/>
      • 상품 정보, 현재 재고와 판매수량 확인가능<br/>
      • 재고칸을 클릭해서 재고를 수정 및 품절, 판매종료 처리 기능<br/>
      • 상품을 복수선택 하여 상품삭제 버튼을 통해 삭제할 수 있습니다. <br/>
  C. 주문 목록 조회 <br/>
      • 배송 상태 클릭해 배송처리 => 사용자가 리뷰 작성 가능 <br/>

#### 통계 및 주문 분석 Part.

1. 통계 및 주문 분석 <br/>
  A. 판매자 메인 화면 대시보드 <br/>
      • 판매 현황, 배송 현황, 신규 게시글 등 요약 정보 조회 <br/>
  B. 통계 - 판매 분석화면 <br/>
      • 카테고리 별 주문량 요약 <br/>
      • 판매량 분석 및 요약 <br/>
      • 판매량 추이 그래프 <br/>
  C. 통계 – 사용자 행동 양상 분석 <br/>
      • 별점수,리뷰수,사용자성별등으로유의미한 통계 도출 <br/>
